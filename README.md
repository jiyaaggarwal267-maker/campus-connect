# CampusConnect - College Event Management Portal

## Project Overview

CampusConnect is a college event management platform where students can explore upcoming events, register for events, manage their registrations, and receive announcements.

The platform provides separate access for:

* Students
* Admin

## Features

### Student Features

* Create student account
* Login securely
* View available events
* Register for events
* View registered events
* Cancel event registrations
* View announcements

### Admin Features

* Admin-only dashboard
* Create new events
* Upload event banners
* Delete events
* Create announcements
* Delete announcements
* View event statistics

## Technology Stack

### Frontend

* React.js
* TypeScript
* Tailwind CSS
* React Router
* Axios

### Backend

* Node.js
* Express.js
* TypeScript
* MongoDB
* JWT Authentication

### Other Services

* MongoDB Atlas
* Cloudinary (Event banner storage)

## How to Run Locally

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```

## Deployment

The application is deployed with:

* Frontend: Vercel
* Backend: Render
* Database: MongoDB Atlas
